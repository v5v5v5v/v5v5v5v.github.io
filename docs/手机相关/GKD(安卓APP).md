## GKD matches规则
1.   `*` 表示任意控件，可以是 TextView，Button，ImageView 等等

```html
// * 可以省略
Button[text^="跳过"] 简化为：[text^="跳过"]
```

2.   模糊匹配：TextView[text^="跳过"]

     -   ^= 表示匹配以 xxx 开头的字符串

     - = : 匹配与 xxx 相同的字符串

     - $= : 匹配以 xxx 结尾的字符串

     - *= : 匹配包含 xxx 的字符串

     - != : 匹配与 xxx 不同的字符串

     - !^= : 匹配不以 xxx 开头的字符串

     - !$= : 匹配不以 xxx 结尾的字符串

     - !*= : 匹配不包含 xxx 的字符串

3.   比较操作符

     ```bash
     > : 大于
     < : 小于
     = : 等于
     != : 不等于
     >= : 大于等于
     <= : 小于等于
     ```

4.   关系操作符
     -   兄弟节点（+，-）：[text="取消"] + [text="更新"]，`+`两侧必须要有空格
     -   父子节点（>，<）

5.   @ 手动指定某个属性选择器作为目标控件，将符号 @ 放置在属性选择器前面即可，例如：
     `@[text="取消"] + [text="更新"]`
