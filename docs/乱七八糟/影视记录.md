## 笔记

1.   编码标准

     -   ＡＶＣ = H.264：兼容性拉满，但文件大，适合老设备播放

     -   ＨＥＶＣ = H.265：相比AVC，画质体积减半，老旧设备可能播不了

2.   字幕标识

     -   ＧＢ：简体
     -   ＣＨＴ：繁体（台版）

3.   画质排名：

     1.   发烧无损梯队

          |      画质      |                         说明                         |
          | :------------: | :--------------------------------------------------: |
          |  BDMV / BDISO  |  无损完整碟片，音画全部原生，带花絮多音轨，体积最大  |
          |  BluRay REMUX  | 原盘无损提取，删掉花絮，画质和原盘完全一致，体积更小 |
          | BluRay / BDRip |        蓝光源二次压制，清晰度高，日常收藏主流        |

     2.   流媒体梯队（院线没出蓝光前首选）

          |  画质  |                            说明                            |
          | :----: | :--------------------------------------------------------: |
          | WEB-DL | 平台官方源码直扒，无二次压缩，干净无水印，流媒体里画质顶峰 |
          | WEBRip | WEB-DL 再转码 / 录屏，经过二次压缩，细节、色彩不如 WEB-DL  |
          | HDRip  |       蓝光压制精简版，码率低，体积小，画质弱于 BDRip       |

     3.   老光碟源、最差枪版

          DVDRip：DVD 碟压制，最高仅 480P，清晰度差
          DVDScr / R5：院线预览碟，色彩偏暗、有水印
          TC：影院拷贝机导出，亮度低、色偏
          TS / CAM：手机影院偷拍，抖动、杂音、黑影，画质最差
